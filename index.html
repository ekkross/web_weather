What is the project about?

I created a Python script to visualize the weather of 500+ cities across the world. I used the Python library citipy and the OpenWeatherMap API to create a representative model of weather across world cities.

I built scatter plots to show the following relationships:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
